feat(interview): implement AI interview runtime, session engine and analytics report system

Overview

This PR introduces the complete Interview Runtime and AI-powered Report Engine for PrepPilot AI. It transforms the platform from a static interview generator into a full interview simulation system with persistent runtime sessions, AI evaluation, analytics, and comprehensive performance reports.

The implementation is divided into three major milestones:

Phase 6.0 – Interview Runtime Foundation
Phase 6.1 – Runtime Session History
Phase 6.2 – Interview Report & Analytics Engine (Commercial Polish)
